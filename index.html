Q3. Write the code to access element which is having class as "box"

Answer: The code to access element is, document.getElementsByClassName("box")[indexnumber];  Click Here for implementation

Here, getElementsByClassName returns a HTML Collection which is in the form of an array. Specifying the index number is must to access the element since, many elements can have same class names. This can also be done using Query selector also.But, query selector always returns the first element that matches.
